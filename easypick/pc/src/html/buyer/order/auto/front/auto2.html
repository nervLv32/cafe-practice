엑셀 업로드

엑셀파일을 업로드하여 주문을 등록할 수 있습니다.

  • 1. 엑셀의 등록 양식을 설정해주세요
  • 2. 설정된 파일에 주문정보를 입력해주세요
  • 3. 주문정보가 입력된 파일을 선택 후
  • 4. '주문등록하기' 버튼을 클릭해 주세요

주문정보가 잘못된 경우 주문을 할 수 없습니다.

  • "오류" 상태를 확인하시어 주문을 수정해주세요

재확인 요청 엑셀 업로드 방법

  1. [이지픽 > 주문관리 > 오늘입고상품] 또는 [이지픽 > 주문관리 > 검수대기상품]에서 [엑셀다운로드 > 재확인 요청 양식]을 다운로드 받으세요.
  2. 주문 내용을 확인하여 재확인 요청이 필요한 품목은 재확인 요청 열에 요청내용을 기입하고 작성하세요.
  3. [이지픽 > 주문관리 > 검수대기상품]에서 [재확인 요청 업로드] 버튼을 클릭하세요.
  4. 본 팝업에서 요청내용이 입력된 엑셀 파일을 [파일찾기] 버튼을 통해 불러온 뒤 업로드해주세요.

유의사항

  • [재확인 요청 양식]인 경우에만 업로드 시 매칭이 가능합니다.
  • 최대 1,000건 까지 등록가능합니다.
  • xlsx 또는 xls 파일 형식만 업로드가 가능합니다.
  • 업로드 후 실패 건이 있는 경우 재확인요청 [적용]이 불가합니다.
    실패건의 오류상태를 확인하여, 엑셀을 수정하여 다시 업로드해주세요.

- 첨부파일: xls, xlsx 확장자, 최대 1,000건

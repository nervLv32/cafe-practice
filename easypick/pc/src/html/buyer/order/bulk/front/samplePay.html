엑셀 업로드

엑셀파일을 업로드하여 상품을 등록할 수 있습니다.

  • 1. 먼저 엑셀 등록 양식을 설정해 주세요.
  • 2. 엑셀 양식을 다운로드 받은 후 양식에 맞춰 주문 정보를 입력해 주세요.
  • 3. 찾아보기 버튼을 클릭하여 주문 정보가 입력된 파일을 선택해 주세요.
  • 4. '엑셀업로드' 버튼을 클릭해 주세요.

주문 정보가 잘못된 경우 주문을 진행할 수 없습니다.

  • 엑셀 업로드 후 오류 상태를 확인하시어 주문 정보를 수정해 주세요.

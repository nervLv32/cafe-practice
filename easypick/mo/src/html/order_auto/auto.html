주문수량
최근 1주일간 주문수량 평균
가용재고
정상재고에서 배송예정수량을 차감한 실제 출고가 가능한 재고 수량
입고대기
신규 발주 후 아직 입고되지 않은 수량의 합계
배송전취소
주문 취소/반품 등 배송전 취소가 확정되어 재고에 추가 예정인 수량
필요수량
가용재고 + 입고대기 수량이 0보다 작은 경우 해당 수량만큼을 필요수량으로 계산
발주예측수량
입력된 데이터를 기준으로 이지픽에서 계산된 예측 수량
(주문수량 - 가용재고 - 입고대기 - 배송전취소 수량)

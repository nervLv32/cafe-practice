자동주문 가이드

자동주문이란?
재고솔루션에서 데이터를 불러와서 직접 주문을 하거나, 예약해둔 시간에 자동 주문을 진행할 수 있는 편의 기능입니다.
(PC버전에서 엑셀 파일을 직접 업로드할수도 있습니다.)
자동주문 이용가이드
  • STEP 01. 품목 등록
  • STEP 02. SET 설정
  • STEP 03. 시나리오 설정
  • STEP 04. 전표 관리
  1. 자동주문 페이지에서 [품목불러오기] 버튼을 클릭하여 품목 정보를 가져오거나, 엑셀 양식을 다운로드 받아 데이터를 입력 후 엑셀 파일을 업로드해 주세요.
  2. 품목불러오기 진행 시, 재고솔루션에서 불러올 품목의 옵션을 먼저 설정 후 저장하기 버튼을 클릭해 주세요.
  3. 자동주문 페이지에서 [품목불러오기] 버튼 클릭 시, 저장된 옵션에 따라 품목 불러오기가 가능합니다.
  1. SET 목록 화면에서 [SET추가] 버튼 또는 등록되어있는 샘플을 복사해 주세요.
  2. 팝업에서 원하는 필터와 실행조건을 추가해 주세요.
  3. SET 설정 완료 후 시나리오를 설정해 주세요.
  1. [시나리오추가] 버튼을 클릭해 주세요. 기존에 등록된 시나리오를 수정하시려면 [수정] 버튼을 클릭해 주세요.
  2. 시나리오 실행 시 사용할 SET를 선택해 주세요.
  3. 시나리오 설정 완료 후 [예약 설정] 버튼을 클릭하여 자동주문 예약설정을 진행해 주세요. 예약을 하지 않고 수동 실행도 가능합니다.
  1. 자동주문 페이지에서 주문 완료까지 진행한 주문건에 한하여 재고플러스로 전표를 자동 전송하거나 엑셀 양식을 다운로드 받을 수 있습니다.
  2. 시나리오 설정 화면>전표자동전송 항목>'전송'에 치크 시, 별도의 작업 없이 주문완료 시점에서 재고플러스로 전표가 자동 전송됩니다.
  3. 전표 양식 파일은 주문완료 후 이지픽 PC 주문내역 상세페이지에서 다운로드 받을 수 있습니다.

자동주문설정

시나리오 실행만 하기
설정해둔 옵션대로 시나리오가 실행된 후, 자동주문 페이지 품목리스트에 결과를 반영합니다.
품목리스트를 확인 후 주문을 진행하거나 편집할 수 있습니다.
자동 주문하기
시나리오 예약 설정 화면에서 예약해둔 요일과 시간에 자동으로 주문이 진행됩니다.
  • 자동주문 시, 잘못된 설정으로 인한 문제는 당사에서 책임지지 않습니다.
  • 설정한 자동주문 최대 금액을 초과할 경우 주문이 진행되지 않습니다.

전송
전송함 선택 시 주문완료 시점에 재고플러스로 전표가 자동으로 전송됩니다.

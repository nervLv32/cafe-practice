자동 제작마감 안내

  • '자동 제작마감' 기능은 자동 제작마감[ON] 설정 시,
    입고된 제작상품을 자동으로 '제작마감' 처리해주는 기능입니다.
    (거래처에게 미송 발주가 나가지 않습니다.)
  • '처리중'인 상품이 있는 경우,
    [ON/OFF] 설정 변경 시 다음 발주 건 부터 반영되며,
    '제작마감' 처리 후 마감 취소는 불가하오니 참고 바랍니다.

제작상품 마감 안내

  • 거래처에 처음 발주되기 전에는, 주문취소가 가능합니다.

    폐기처리 되니, 유의하시기 바랍니다.
